## 🚀 &nbsp; Run Locally

```bash
# 1. Clone
git clone https://github.com/Santosh-A-05/portfolio.git

# 2. Move into the folder
cd portfolio

# 3. Open in browser
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Or serve it with a quick dev server:

```bash
# Python 3
python -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.
